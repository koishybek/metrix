## 1. Product Overview
Metrix — это личный кабинет для клиентов, позволяющий проверить статус умных водосчётчиков по серийному номеру прибора. Продукт решает проблему быстрого доступа к данным о водосчётчиках без необходимости звонков в поддержку.

Целевая аудитория — владельцы квартир с установленными умными водосчётчиками SmartMetrix. Продукт предоставляет мгновенный доступ к актуальной информации о показаниях и статусе прибора.

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Guest User | No registration required | Can check meter status by serial number |

### 2.2 Feature Module
Приложение Metrix состоит из следующих основных страниц:
1. **Главная страница**: поле ввода серийного номера, кнопка проверки статуса, логотип и слоган.
2. **Результат проверки**: карточка с данными о счётчике, история показаний, кнопка проверки другого номера.

## 3. Core Process
**Поток пользователя:**
1. Пользователь открывает главную страницу Metrix
2. Вводит серийный номер водосчётчика в текстовое поле
3. Нажимает кнопку "ПОКАЗАТЬ СТАТУС"
4. Система показывает лоадер и делает GET-запрос к API SmartMetrix
5. При успешном ответе отображается карточка с данными о счётчике
6. Пользователь может развернуть историю показаний или проверить другой номер
7. При отсутствии данных показывается блок с причинами и контактами поддержки

## 4. User Interface Design

### 4.1 Design Style
- **Основные цвета**: синий (#0066CC / #1E90FF) для основных элементов, зелёный (#28a745) для успеха, красный (#dc3545) для ошибок/оффлайн, жёлтый/оранжевый для предупреждений
- **Шрифт**: Inter или системный sans-serif, крупные размеры текста для мобильных устройств
- **Кнопки**: Крупные, закруглённые, на всю ширину на мобильных, контрастные цвета
- **Фон**: Белый или светло-серый (#f8f9fa) для карточек
- **Иконки**: Минималистичные, линейные стили, иконка капли воды для логотипа
- **Тени**: Лёгкие тени для карточек и кнопок (shadow-sm)

### 4.3 Responsiveness
- **Мобильный-first подход**: Основная разработка ведётся под мобильные устройства (iPhone/Android)
- **Адаптивность**: Приложение полностью адаптивно под все размеры экранов
- **Touch-оптимизация**: Крупные кнопки и поля ввода для удобства работы пальцами
- **Breakpoints**: 320px (mobile), 768px (tablet), 1024px (desktop)
- **Максимальная ширина контента**: 600px для формы ввода, 800px для карточки результата
